 ### Install yarn
```
npm i yarn -g
```

### It is recommended to use npm Taobao mirror in China
```
npm config set registry http://registry.npm.taobao.org/
```

### Use yarn to install dependent packages
```
yarn
```

### Start the server
```
yarn start
```
